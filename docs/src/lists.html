 shape_intersections(+NamedShapes, -Intersections) is det
Encontra as intersecções entre todos os NamedShapes. Um NamedShape é do tipo Name:Shape e um intersecção é Name-Name. 
?- shape_intersections([a:circle((0, 0), 1), b:square((1, 1), 1), c:circle((2, -2), 1)], X).
X = [a-b].
