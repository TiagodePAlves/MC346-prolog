 shape(+Name, +Shape)
Verdade se existe uma figura Shape associada a Name.

Fato dinâmico.

 intersection(?NameA, ?NameB)
Verdade se existe uma figura com nome NameA e outra com NameB e elas têm intersecção não vazia.
 ordered_intersection(?NameA, ?NameB)
Parecido com intersection/2, mas para ser verdade também é preciso que NameA seja lexigraficamente menor que NameB.
 database_solver(+Figures, -Intersections, -Length) is det
Resolve o problema para as figuras dadas e resolve o tamanho da solução também. 
?- database_solver([circ(a, 0, 0, 1), quad(b, 1, 1, 1), circ(c, 2, -2, 1)], N, X).
N = 1,
X = [a-b].
